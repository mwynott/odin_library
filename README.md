# odin_library
TOP library project


This library comes preloaded with a five books so it would be easier to test. 
There is a form for adding a new book to the existing library, which assigns a
unique id to each entry by way of the "Add New Book" button. Upon completion
the library is generated for viewing (plus the six existing books). There is a
checkbox for whether or not one has read the book as well as a delete button
which removes the entry by way of its unique id. All in all this was a tough
little project and it took extra reading to get back up to speed with JS
after so much time off during the very long Advanced HTML/CSS unit.
